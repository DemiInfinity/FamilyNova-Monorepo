# FamilyNova Web Application

Web application for FamilyNova that brings together all platforms and provides a unified experience.

## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```

## Features

- Landing page with information about FamilyNova
- Kids portal (login/registration)
- Parents portal (login/registration)
- Responsive design
- Kid and parent-friendly UI

## Technology Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: React Query (planned)

## API Integration

The web app connects to the backend API at `http://localhost:3000/api` (configurable via environment variables).

## Design

- Uses the same design system as mobile apps
- Kid-friendly colors for kids portal
- Professional colors for parents portal
- Responsive and accessible
